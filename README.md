# Laravel Pages

## Description
Laravel Pages est un petit projet pour apprendre à créer des pages avec Laravel et Blade.  
Il contient deux pages principales : Accueil et À propos, avec un layout commun utilisant Bootstrap.

## Prérequis
- PHP >= 8.1
- Composer
- Git
- Navigateur web
